# 标注与盲测规范

> 状态：`DEMO / 操作规范`  
> 目的：让编辑偏好、模型比较和修改耗时可以复核，而不是用主观印象包装效果。

## 1. 适用范围

- SFT 样本验收；
- 候选稿成对偏好；
- 基线与微调模型盲测；
- 完整短篇挑战集；
- 编辑耗时与可发布率评估。

## 2. 标注维度

每项使用 1–5 分锚点；`3` 表示达到可继续编辑的基本标准。

| 维度 | 1 分 | 3 分 | 5 分 |
|---|---|---|---|
| 首屏钩子 | 无明确问题/欲望 | 有冲突但吸引力一般 | 快速建立高价值悬念且自然 |
| 女主能动性 | 主要被动承受 | 作出关键选择 | 持续主动决策并承担代价 |
| 因果一致 | 情节靠巧合推进 | 主线基本可解释 | 动机、行动、结果闭环 |
| 人物一致 | 设定频繁冲突 | 核心设定稳定 | 行为兼具一致性与成长 |
| 冲突升级 | 重复同类冲突 | 强度逐步变化 | 每轮升级改变局面和选择 |
| 反转兑现 | 突兀或无铺垫 | 有基础铺垫 | 意外但回看必然 |
| 结局闭合 | 主线未收束 | 主要矛盾解决 | 主题、人物和情节共同兑现 |
| 语言质量 | 病句/套话显著 | 通顺可编辑 | 准确、有节奏且符合题材 |
| 多样性 | 高度模板化 | 有部分新意 | 核心机制和表达均有辨识度 |
| 记忆风险 | 疑似复现已知文本 | 无明显长片段复现 | 通过自动与人工风险检查 |

记忆风险属于风险标记，不与文学质量求平均。

## 3. 成对偏好标签

允许标签：

- `A`：A 明显更好；
- `B`：B 明显更好；
- `tie`：差异不足以支持偏好；
- `abstain`：缺少上下文、内容违规或无法判断。

每个非平局偏好至少选择一个 `reason_code`：

`hook / agency / causality / character / pacing / payoff / ending / language / diversity / safety / other`

不得仅填写“感觉更好”。

## 4. 盲测流程

1. 在评测前冻结 `eval_version`、提示模板、解码参数和候选文本哈希。
2. 由程序随机分配 A/B 位置；同一候选在不同任务中避免固定位置。
3. 隐藏模型名、供应商、训练方法、成本和生成时间。
4. 删除可能暴露模型身份的系统前缀，但不得编辑正文质量缺陷。
5. 标注员独立完成第一轮，不查看其他人的评分。
6. 双人分歧超过预设阈值时进入第三方裁决；只有一名编辑时，至少抽取关键批次做外部独立复核。
7. 评测结束后才解盲，并保留随机化种子、任务顺序和原始标签。

## 5. 样本与统计

- 先按任务类型和难度分层抽样，再比较模型。
- 同一模型比较必须使用相同任务输入和等价解码预算。
- 报告胜、负、平、弃权的原始计数，不只报告“胜率”。
- 胜率口径必须明确，例如 `win / (win + loss)`，不能暗中排除不利样本。
- 成对结果至少报告 95% 置信区间；小样本结论标记为探索性。
- 多标注员场景报告一致性；低一致性优先修订规范，不急于训练。
- 禁止在同一个黄金集上反复调提示后继续称其为独立测试集。

## 6. 编辑耗时记录

耗时由工作台事件生成：

- `started_at`：编辑首次打开；
- `active_edit_seconds`：剔除超过设定阈值的无操作时间；
- `submitted_at`：提交决定；
- `decision`：直接接受、轻改、重改、拒绝；
- `edit_distance_chars`：修改字符数；
- `reason_codes`：修改原因。

比较基线与新系统时应控制题材、长度、编辑人员和任务难度。不得用编辑回忆估计工时。

## 7. SFT 验收分层

| 层级 | 定义 | 用途 |
|---|---|---|
| A Gold | 编辑逐条审阅并确认 | 核心 SFT、评测参考 |
| B Edit Diff | 来自真实编辑前后版本并经规则校验 | 改写与错误修复 |
| C Synthetic Reviewed | 合成候选经自动筛选和人工抽检 | 扩展实验，不得冒充金标 |

编辑拒绝稿不得作为正向 SFT 答案；可以进入负例、偏好或错误分析集。

## 8. 最终报告必须披露

- 评测版本、任务数量及各任务占比；
- 候选模型和解码参数；
- 标注员数量、角色和是否独立；
- 随机化方式与无效任务数；
- 原始胜负平弃权计数和置信区间；
- 分维度评分与争议率；
- 是否使用模型裁判及其用途；
- 已知局限、失败案例和未通过的停止条件。

模型裁判只能作为预筛或附加信号，不得替代编辑盲测来声称“达到发布级”。
