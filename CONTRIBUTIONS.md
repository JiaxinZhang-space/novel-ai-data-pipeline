# 作品集贡献与声明

## 维护者与角色

- 公开维护者：[`JiaxinZhang-space`](https://github.com/JiaxinZhang-space)
- 角色：AI 数据工程作品集维护者

## 当前可核验贡献

维护者对本公开仓库承担以下工作：

1. 将小说生成业务问题拆成权利、规范化、去重、切分、数据派生、质量和血缘组件；
2. 维护可重复运行的 Python 管道和输出目录安全保护；
3. 设计并验证作者/作品级集合隔离；
4. 生成 SFT、偏好对和隔离评测三类数据契约与合成样例；
5. 维护 Dataset Card、权利/撤回台账、数据字典、实验和证据模板；
6. 运行自动化测试、校验和对账与公开发布安全检查；
7. 解释已实现能力、目标规范和生产缺口之间的边界。

## 当前不能声明的成果

- 处理过真实授权小说或真实编辑修改记录；
- 训练、微调或上线了小说生成模型；
- 在真实盲测中优于基线；
- 降低了真实编辑耗时；
- 获得真实用户、发布量或收入；
- 代表朋友、公司、客户或任何第三方团队。

若将来存在真实合作，必须使用新的私有项目保存原始证据；公开仓库只登记经授权且经复核的聚合声明，不回填或伪造历史 Git 记录。

## AI 辅助开发

本仓库使用 AI 编码助手进行方案讨论、代码和文档起草、审计与测试加速。维护者负责最终的需求判断、运行验证、内容取舍、公开发布和面试解释。面试时可以表述为“AI-assisted development”，不能把工具生成过程包装成未经本人理解和验证的手写经历。
